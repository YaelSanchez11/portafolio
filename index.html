Proyectos

Página de Recetas

Una aplicación web diseñada para amantes de la cocina, donde los usuarios pueden explorar una amplia variedad de recetas categorizadas por tipo de comida. Incluye funciones para guardar recetas favoritas y agregar comentarios. Además, cuenta con un chatbot de IBM Watson que brinda recomendaciones personalizadas y responde preguntas relacionadas con ingredientes, preparación y tiempos de cocción. Ideal para quienes buscan inspiración en la cocina o desean mejorar sus habilidades culinarias.

Página para Organizar Tareas

Un organizador de tareas intuitivo desarrollado para estudiantes y profesionales. Permite crear, editar y eliminar tareas, asignar fechas de vencimiento y establecer prioridades. Su interfaz minimalista facilita la gestión del tiempo y mejora la productividad. Además, incluye una sección de seguimiento de progreso, recordatorios automáticos y la opción de compartir tareas con otros usuarios. Una herramienta efectiva para cumplir objetivos diarios y a largo plazo.

PAGINA CON INICIO DE SESION

Página cuyo principal objetivo es permitir el registro de usuarios para que puedan crear una cuenta personal e iniciar sesión de manera segura. Este sistema de autenticación garantiza que solo usuarios autorizados puedan acceder a las funciones y contenidos exclusivos de la plataforma.
